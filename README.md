# Subscription Calculator

### Here is a mock subscription calculator. It provides the user with the cost of plans for different subscription types and durations.

### Built using:
####- HTLM5
####- CSS3
####- JavaScript

### This is an exercise in the use of event listeners, event handlers, function expressions, and callback functions to calculate the numbers and return a message.
